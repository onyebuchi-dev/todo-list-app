# To-Do List App

A simple web application to manage daily tasks built with JavaScript, HTML, and CSS.

## Features
- Add new tasks
- Delete tasks
- Interactive and user-friendly interface

## Technologies Used
- JavaScript
- HTML5
- CSS3

## How to Use
1. Open `index.html` in your browser.
2. Type a task in the input box.
3. Click "Add Task" or press Enter.
4. Delete tasks by clicking the "Delete" button.
